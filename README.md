<h1>Uni-pulse Mobile Application</h1>
<p>Build with react-native, expo, typescript, nativewind (tailwind)</p>

<hr />

<h3>Installation & setup guide</h3>

```bash
git clone https://github.com/itsmenoahpoli/unipulsemobile.git

cd unipulsemobile

npm install

npm run start
```

<small>Please scan the QR code with your Expo Go app on your phone.</small>
